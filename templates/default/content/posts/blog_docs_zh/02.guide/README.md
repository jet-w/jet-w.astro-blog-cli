# Markdown 基础

Markdown 是一种轻量级标记语言，让格式化内容的编写变得简单直观。

## 标题

使用 `#` 符号表示标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

::: tip 最佳实践
- 每页只使用一个 `# 一级标题`（你的标题）
- 不要跳级（不要从 `##` 跳到 `####`）
- 保持标题简洁
:::

## 文本格式

### 基本样式

| 语法 | 结果 | 用途 |
|------|------|------|
| `**粗体**` | **粗体** | 重要术语 |
| `*斜体*` | *斜体* | 强调 |
| `~~删除线~~` | ~~删除线~~ | 已弃用的内容 |
| `` `代码` `` | `代码` | 行内代码 |

### 组合样式

```markdown
***粗斜体***
**粗体中有 `代码`**
```

## 链接

### 基本链接

```markdown
[链接文本](https://example.com)
[带标题的链接](https://example.com "悬停文本")
```

### 内部链接

链接到其他文章：

```markdown
[查看指南](./02-containers)
[返回首页](/)
```

## 图片

### 基本图片

```markdown
![替代文本](/images/photo.jpg)
```

### 带说明的图片

```markdown
![美丽的日落](/images/sunset.jpg)
*海上美丽的日落*
```

### 图片尺寸

使用 HTML 进行更多控制：

```html
<img src="/images/photo.jpg" alt="照片" width="300" />
```

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 嵌套项目
  - 另一个嵌套
- 项目三
```

结果：
- 项目一
- 项目二
  - 嵌套项目
  - 另一个嵌套
- 项目三

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 任务列表

```markdown
- [x] 已完成的任务
- [ ] 待完成的任务
- [ ] 另一个任务
```

- [x] 已完成的任务
- [ ] 待完成的任务
- [ ] 另一个任务

## 引用

```markdown
> 这是一段引用。
> 可以跨越多行。

> 嵌套引用：
>> 也是可以的
```

> 这是一段引用。
> 可以跨越多行。
